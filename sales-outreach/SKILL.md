# Sales Outreach Vorlage

**WICHTIG — Vor Verwendung dieser Vorlage IMMER laden:**
1. `memory/AboutRetoBaettig.md` — Persönlicher Hintergrund, Kommunikationsstil, Werte
2. `memory/AboutCudos.md` — Firmen-Infos, Mission, Vision, Cudos Trail, Zahlen

## Struktur einer Outreach-Email

### 1. Betreffzeile
```
Kontaktaufnahme
```

### 2. Einleitung (Persönlich + Unternehmensprofil)
```
Guten Tag Herr [Name],

Mein Name ist Reto Bättig, ich bin CEO der Cudos AG, einem innovativen
schweizer Software Dienstleister mit >60 top qualifizierten Ingenieuren,
Projektleitern, Data Scientists, Architekten etc.
```

### 3. Positionierung (KI-Value-Proposition)
```
Stellen Sie sich vor, Individualsoftwareentwicklung würde kostenlos — da
durch KI geschrieben. Wie würde das Ihre IT- und
Digitalisierungslandschaft verändern? Wie würde es Ihre Make oder Buy
Entscheide beeinflussen? Welche Prozesse würden Sie jetzt endlich verbessern
und digitaler machen?
```

### 4. Value Proposition (Nicht nur Chatbot)
```
Wir haben uns viele Gedanken darüber gemacht, wie man KI-geschriebene
Software managen kann, damit sie sicher, wartbar und hochverlässig ist und
schnell echten Business-Nutzen und Einsparungen bringt. Es ist NICHT ein
KI-Angebot, sondern echte, funktionierende, reproduzierbare, sichere
Software die dank KI im Prozess günstiger hergestellt werden kann. Gerne
kann ich Ihnen auch schon konkrete Resultate und Beispiele zeigen.
```

### 5. Call-to-Action (Konkret + Zeitlich begrenzt)
```
Es würde mich so oder so sehr freuen, Sie kurz kennen lernen zu dürfen und
Ihre Meinung zu unseren Ideen zu erfahren. 25min wäre ein super Start!

25min Meeting mit Reto: [Calendly-Link]

Vielen Dank im Voraus und freundliche Grüsse,
Reto Bättig
```

## Key Elements für Ansprachen

1. **Persönliche Einleitung** mit Name, Rolle, Unternehmensprofil
2. **Provokante Frage** (Was wäre wenn...)
3. **KI-Differenzierung** (nicht nur Chatbot, sondern echte Software)
4. **Sicherheit/Reproduzierbarkeit** betonen
5. **Konkreter CTA** mit kurzem Zeitfenster (25min)
6. **Calendly-Link** einfügen
